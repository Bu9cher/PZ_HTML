HTML

HTML — стандартизована мова розмітки документів для перегляду вебсторінок у браузері.

Браузери отримують HTML документ від сервера за протоколами HTTP/HTTPS або відкривають з локального диска, далі інтерпретують код в інтерфейсу

В HTML можна вбудовувати програми, написані на скриптових мовах, наприклад JavaScript, які впливають на поведінку та вміст вебсторінок.

Основні теги HTML:

  • html – огортає весь HTML-документ.
  • head – містить службову інформацію про сторінку.
  • title – визначає заголовок сторінки у вкладці браузера.
  • body – містить основний вміст веб-сторінки.

Короткі факти HTML:

  1. Основна структура – документ складається з head і body.
  2. CSS та HTML – використовується для стилізації сторінок.
  3. JavaScript у HTML – додає інтерактивність.
  4. Форми в HTML – дозволяють вводити та відправляти дані.
